## 🚀 How to Use

1. Download the `.html` file
2. Double-click it — it opens in any browser (Chrome, Safari, Edge)
3. Start adding your deals!
4. Click **Export CSV** anytime to open your data in Google Sheets

> 💡 Your data stays on **your computer only**. Nothing is uploaded or shared. Each person who downloads this file gets their own completely private copy.

## 🔒 Privacy

This tracker runs 100% in your browser. There is no server, no database, and no internet connection between users. Your deals and client data never leave your device.
